# env-linux

Shell scripts to prepare my computer after formatting.

### Usage

```
wget https://raw.githubusercontent.com/georgemaia/env-linux/master/SCRIPT.sh
chmod +x SCRIPT.sh
sudo ./SCRIPT.sh

where SCRIPT is the name of the script.
```
